# Code of Conduct

AutoShift is built on [Open Cluster Management](https://open-cluster-management.io/), a CNCF project. This community abides by the [CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md).

To report a Code of Conduct violation, open a private [Security Advisory](https://github.com/auto-shift/autoshiftv2/security/advisories/new) (this keeps the report confidential from the public). If the violation involves a maintainer, escalate directly to the [CNCF Conduct staff](https://www.cncf.io/conduct/). All reports will be kept confidential.
